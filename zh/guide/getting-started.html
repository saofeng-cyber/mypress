# 快速上手

提示

node 版本应不小于 16，pnpm 版本应不小于 6

如果您还没安装 pnpm，请执行下面命令进行安装（mac 用户遇到安装报错请在命令前加上 sudo）

npm install -g pnpm

如果您需要安装多个 node 版本环境，请参考 Node.js版本管理工具open in new window

如果您觉得安装平台依赖慢，请参考 npmmirroropen in new window

# 本地开发

安装依赖

pnpm install

启动平台

pnpm dev

安装一个包

pnpm add 包名

安装一个包

pnpm remove 包名
